# 脚本示例

## 重启进程

```sh
# 该脚本用于根据 cmdline 杀死进程，再用 nohup 重启。
# 使用之前，只需修改第一行的 CMD 变量
# 使用时，只需执行 sh restart.sh

CMD="ping localhost"

PID=`ps auxf | grep "$CMD" | grep -v grep | awk '{print $2}' | xargs `
if [ $PID ]; then
    kill -9 $PID
    echo "The process is running, kill it by pid: $PID"
    sleep 3
fi

nohup $CMD &>> nohup.out &
echo "Started process: $CMD"
```
