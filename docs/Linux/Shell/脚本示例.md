# 脚本示例

## 重启进程

```sh
# 该脚本用于根据启动命令杀死进程，再用 nohup 重启。
# 使用之前，需要将 CMD 变量设置为进程的启动命令
# 使用时，只需执行 sh restart.sh

WORK_DIR=`dirname $0`
cd $WORK_DIR

CMD="ping localhost"

PID=`ps auxf | grep "$CMD" | grep -v grep | awk '{print $2}' | xargs `
if [ $PID ]; then
    kill -9 $PID
    echo "The process is running, kill it by pid: $PID"
    sleep 3
fi

echo > nuhup.out            # 每次重启时清空日志文件
nohup $CMD &>> nohup.out &
echo "Started process: $CMD"
```

## 批量打包文件

```sh
file_list=`ls *.mp4`
for f in $file_list
do
    zip $f.zip $f -P '******'
    rm -f $f
done
```
