# 插件工具

## bash-completion

：用于实现命令的自动补全。
- 执行 `yum install bash-completion` 安装，然后重启 shell 即可。
- /usr/share/bash-completion/completions 目录下保存了各种命令的自动补全脚本，

## 常用脚本

### 根据 cmdline 终止进程

```sh
CMD=python
PID=`ps auxf | grep $CMD | grep -v grep | awk '{print $2}'`
if [ $PID ]; then
    kill -9 $PID
else
    echo "Process doesn't exist!"
fi
```
