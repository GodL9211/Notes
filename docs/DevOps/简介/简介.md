# 简介

DevOps：通过一些策略、工具，促进开发人员与运维人员之间的沟通、协作，从而提高整体的工作效率。
- 通常采用“敏捷开发”的策略，缩短软件从开发到发布的周期。
- 通常采用很多自动化工具，取代某些传统的人工作业，提高工作效率。

## 配置管理工具

- Ansible
- Puppet
- Chef
- Saltstack
- Fabric

## 服务发现工具

- Zookeeper
- etcd
  - k8s就是采用etcd存储服务的信息。
  - 一种方案：用etcd存储服务的信息，用registrator注册服务并保存到etcd中，用confd读取etcd并生成配置文件。
- Consul
- Eureka

## 运维监控工具

- Nagios
- Zabbix
- Pandora FMS
