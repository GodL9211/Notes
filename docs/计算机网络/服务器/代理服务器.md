# 代理服务器



## shadowsocks

：一个socket5代理服务器。

- 安装：pip3 install shadowsocks
- 用法：
    ```shell
    ssserver -p 3000 -k ****** -m aes-256-cfb -d start  # 启动ss（作为守护进程运行）
    ssserver -d stop                    # 停止

    iptables -F                         # 清空防火墙规则
    tail -f /var/log/shadowsocks.log    # 查看日志
    ```
