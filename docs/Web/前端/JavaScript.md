# JavaScript

：一种网页脚本语言，嵌入HTML中，用于执行一些动作。

## 使用JS的方式

- 在标签`<script>`之内编写JS代码。

    ```html
    <script>
        document.write("<p>Hello World!</p>");
        alert("test...");
    </script>
    ```

- 导入外部的 .js 文件，然后便可以调用它。

    ```html
    <script src="//static.mysite.com/main.js"></script>
    ```
